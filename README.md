# QSmood

Project to track my mood and correlate it with exercise, sleep, day of the week.

## Project setup
```
npm install
```

### Starts development Vue server
```
npm run serve
```

### Compiles for production and for development API server
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Starts server
```
npm run start
```
